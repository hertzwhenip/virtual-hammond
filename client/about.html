Goal

This is the second semester's project in Informatics Lab. Our goal is to develop a web application that allows users to provide a song's RTTTL specification, an organ registry and sound effects to generate an interpretation of a song.

Technology Stack

  • The CherryPy framework is used to serve the JSON API and the client web interface.
  • The matplotlib library is used to create a PNG image of the representation of a song's interpretation.
  • The wave and struct libraries are used to generate the WAV files of a song's interpretation.
  • The jQuery library is used to make AJAX HTTP requests to the API and handle the responses.
  • The Vue.js library is used to write to the DOM from the API responses.
  • The Bootstrap framework is used to build the web interface using HTML and CSS components.
